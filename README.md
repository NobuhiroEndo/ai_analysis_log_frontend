# ai_analysis_log_frontend
### ■ 概要
- 課題のフロントエンド。Lamda+API Gatewayで作成したモックAPIを叩いてレスポンスを取得し、バックエンドにPOSTリクエストとして送る。バックエンド側で登録した内容を、画面に表示する。
- 作成したプログラムは、S3+CloudFrontにデプロイし、独自ドメインアクセスができるようにした。
